# 🔐 Password Manager

Gerenciador de senhas desenvolvido em Python para armazenar e organizar credenciais de forma simples e segura.

## 🚀 Objetivo

Este projeto foi criado com o objetivo de praticar:

* Python
* Tkinter
* SQLite
* CRUD
* Conceitos básicos de segurança

## ✨ Funcionalidades

* ➕ Adicionar credenciais
* 🔍 Pesquisar credenciais
* ✏️ Editar registros
* 🗑️ Remover registros
* 💾 Armazenamento em banco de dados SQLite

## 🛠️ Tecnologias

* Python
* Tkinter
* SQLite
